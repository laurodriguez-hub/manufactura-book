# Conclusiones

En esta sección se presentan las conclusiones generales derivadas del análisis de capacidades de Big Data aplicadas a la industria manufacturera durante el periodo 2019–2020. 

- Las tecnologías de Big Data ofrecen un potencial significativo para optimizar procesos productivos.
- La integración de datos públicos permite una caracterización más completa del sector manufacturero.
- Las capacidades predictivas identificadas fortalecen la toma de decisiones basada en datos.
- La industria está avanzando hacia modelos analíticos más maduros, pero aún existe brecha tecnológica.

Estas conclusiones permiten cerrar el ciclo analítico y justifican la importancia del Big Data en este sector.
